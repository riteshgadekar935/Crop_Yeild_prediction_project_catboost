Crop Yield Prediction Using Machine Learning

Developed an interactive web application using Streamlit and CatBoost Regressor to predict agricultural crop yield in Quintal per Hectare (Q/Ha). The model is trained on historical agricultural data containing country, crop type, year, rainfall, pesticide usage, and average temperature.

The application allows users to input environmental and agricultural factors and instantly receive predicted crop yield values. CatBoost was chosen because of its strong performance with categorical features such as country and crop type without requiring extensive preprocessing.

Machine Learning Workflow
Load and preprocess agricultural dataset.
Convert yield from hectogram/hectare (hg/ha) to quintal/hectare (q/ha).
Train a CatBoost Regression model using historical crop production data.
Build an interactive Streamlit interface for user input.
Generate real-time crop yield predictions.
